Loyihalar

Weather App

HTML, CSS va JavaScript yordamida yaratilgan. Foydalanuvchi o'z shahrini kiritib, real vaqt ob-havo ma'lumotlarini olish imkoniyatiga ega. JavaScript ob-havo xizmatlaridan ma'lumotlarni olib, foydalanuvchiga taqdim etadi.

Pac-Man O'yini

HTML, CSS va JavaScript yordamida yaratilgan klassik Pac-Man o'yini. Foydalanuvchi Pac-Manni boshqarib, labirint ichida harakatlanadi va dushmanlardan qochib, barcha nuqtalarni yeyishga harakat qiladi.

Tic-Tac-Toe O'yini

HTML, CSS va JavaScript yordamida yaratilgan klassik Tic-Tac-Toe o'yini. Foydalanuvchilar bir yoki ikkita o'yinchi rejimida o'ynashlari mumkin. JavaScript yordamida o'yin mantiqi va g'olibni aniqlash funksiyalari amalga oshirilgan.

Dasturlash

HTML

HTML (HyperText Markup Language) veb-sahifalarni yaratish uchun standart belgilar tilidir. Bu barcha veb-kontentlarning asosidir.

CSS

CSS (Cascading Style Sheets) HTML bilan yozilgan hujjatni taqdim qilishni tasvirlash uchun ishlatiladigan uslub sahifalari tilidir. CSS elementlarning ekranlarda qanday ko'rsatilishini tasvirlaydi.

JavaScript

JavaScript veb-sahifalarda murakkab funksiyalarni amalga oshirish imkonini beruvchi dasturlash tilidir. Bu til dinamik yangilanadigan kontent yaratish, multimedia boshqarish, rasmlarni animatsiya qilish va boshqa ko'p narsalar uchun ishlatiladi.

Boshqa Texnologiyalar

HTML, CSS va JavaScript bilan bir qatorda, men Python, Node.js kabi boshqa texnologiyalar va ramkalar bilan ham tanishman.
